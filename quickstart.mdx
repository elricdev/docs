## Add Elric to Your Server

Get started with Elric by inviting the bot to your Discord server and configuring basic settings.

### Invite the Bot

<AccordionGroup>
  <Accordion icon="robot" title="Invite Elric to your server">
    Click the invite link and select your Discord server. Make sure you have "Manage Server" permissions.
    
    [Invite Elric Bot](https://discord.com/oauth2/authorize?client_id=YOUR_BOT_ID&permissions=8&scope=bot)
    
    The bot requires Administrator permissions for full functionality, but you can customize permissions as needed.
  </Accordion>
  <Accordion icon="gear" title="Initial Setup">
    Once invited, run the setup command to configure basic settings:
    
    ```
    /setup
    ```
    
    This will guide you through:
    - Setting up moderation channels
    - Configuring welcome messages
    - Setting up autoroles
    - Enabling logging
  </Accordion>
</AccordionGroup>

### Basic Configuration

<AccordionGroup>

<Accordion icon="shield-halved" title="Configure Moderation">
  Set up moderation features to keep your server safe:
  
  ```
  /config moderation channel:#mod-logs
  /config jail-role:@Jailed
  /config mute-role:@Muted
  ```
</Accordion>
<Accordion icon="hand-wave" title="Setup Welcome System">
  Configure welcome messages for new members:
  
  ```
  /welcome channel:#welcome
  /welcome message:Welcome to the server, {user}!
  /welcome autorole:@Member
  ```
</Accordion>

</AccordionGroup>
